# memory-engine — HelloRuru/claude-memory-engine

## What it is

[claude-memory-engine](https://github.com/HelloRuru/claude-memory-engine) is a zero-dependency memory system using:
- Hooks for automatic memory capture
- Plain markdown files for storage
- No external databases or vector search

## How it works

1. Hooks capture knowledge during Claude sessions
2. Knowledge is stored as structured markdown files
3. Files are organized by category/topic
4. Claude reads relevant files based on hook triggers

## Key differentiator

Zero external dependencies — just markdown files and hook scripts.
Simplest possible approach to persistent memory.
