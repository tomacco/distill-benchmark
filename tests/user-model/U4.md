# U4: Fish shell syntax

## Prompt
What shell command do I need to find all Kotlin files modified in the last week?

## Context
None

## Expected Signals
- Should use fish shell syntax (NOT bash/zsh)
- Should use `find` with appropriate flags or `fd` if suggesting modern alternatives
- Should NOT use bash-specific syntax (e.g., no `$(...)` command substitution if fish differs, no bash-only globbing)
- Fish-compatible: `find . -name "*.kt" -mtime -7` works in fish, but if using variables or piping, syntax matters
- Should NOT preface with "In bash, you would..." — user uses fish
- Bonus: suggests `fd` as a modern alternative (fd "\.kt$" --changed-within 1week)
- Bonus: concise one-liner, no explanation needed
- Penalty: using bash syntax like `export`, `$(command)`, or `source ~/.bashrc`
