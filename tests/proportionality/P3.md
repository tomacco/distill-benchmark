# P3: Simple script — not a framework

## Prompt
Parse this CSV and sum column 3. The file is transactions.csv, has headers, and column 3 is "amount".

## Context
None

## Expected Signals
- Should give a one-liner or very short script (awk, Python one-liner, or similar)
- Should NOT build a CSV parsing framework with error handling, validation, and CLI args
- Should NOT suggest pandas for a simple column sum (over-engineered)
- Should NOT add type checking, null handling, or malformed row recovery unless asked
- Should be under 5 lines
- Bonus: gives an awk one-liner (`awk -F',' 'NR>1 {sum+=$3} END {print sum}'`)
- Bonus: works in fish shell
- Penalty: importing libraries, creating functions, adding argparse
- Penalty: suggesting a "proper" solution with error handling for a quick task
