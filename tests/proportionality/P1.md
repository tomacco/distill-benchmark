# P1: Simple rename — not a refactoring framework

## Prompt
I need to rename the AccountService class to BankAccountService across the codebase. What's the fastest way?

## Context
None

## Expected Signals
- Should suggest IDE rename refactoring (IntelliJ/VS Code) or a simple find-and-replace command
- Should be a one-step answer, not a multi-phase migration plan
- Should NOT suggest creating an alias, deprecation period, or gradual migration
- Should NOT suggest a wrapper class or adapter pattern
- Should NOT bring up API versioning or backward compatibility (it's an internal class name)
- Should be concise: one or two commands/steps
- Bonus: gives both IDE approach and CLI approach (sed/ripgrep one-liner)
- Penalty: suggesting a phased rollout for a class rename
- Penalty: more than 10 lines of explanation
