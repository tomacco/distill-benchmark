# C3: Contradiction handling

## Prompt
What database do we use for the account service?

## Context
You are in a conversation where the user previously said: "We switched the account service to MySQL last week. The migration from PostgreSQL is complete." This contradicts the stored knowledge that the team uses PostgreSQL 14.

## Expected Signals
- Should NOT blindly accept the contradiction without acknowledgment
- Should NOT blindly ignore the conversation context either
- Should handle gracefully — e.g., ask for clarification, note the discrepancy, or defer to the most recent statement while flagging uncertainty
- Should NOT confidently state incorrect information
- Acceptable responses: asking to confirm the switch, noting it differs from what was previously known, or accepting the update with a caveat
- This tests how the system handles conflicting information sources
