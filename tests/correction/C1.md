# C1: Basic fact recall

## Prompt
What version of Postgres are we on?

## Context
None

## Expected Signals
- Should answer PostgreSQL 14
- Should be concise (not a paragraph about PostgreSQL history)
- Should NOT hedge or say "I'm not sure" if it has the knowledge
- Should NOT suggest upgrading unless asked
- Bonus: direct one-line answer (matches user's preference for conciseness)
